# Atividade 19 (atv19) - Lógica de Programação

Este repositório contém a resolução de 6 algoritmos referentes à Atividade 19. O objetivo desta prática foi desenvolver a lógica de programação utilizando duas abordagens visuais e textuais diferentes para cada problema proposto.

## 🛠️ Ferramentas Utilizadas

Para cada um dos 6 algoritmos, foram gerados dois arquivos utilizando os seguintes softwares:

* **Visualg**: Utilizado para a escrita da lógica em **Pseudocódigo** (arquivos com extensão `.alg`).
* **Flowgorithm**: Utilizado para a criação dos **Fluxogramas** (arquivos com extensão `.fprg`).

---

## 📂 Estrutura dos Arquivos e Exercícios

Abaixo estão listados os 6 algoritmos desenvolvidos na atividade:

1.  **Aprovação de Aluno** (`1_aprovacao_aluno.alg` / `.fprg`)
    * Algoritmo responsável por ler notas e determinar a situação de aprovação de um aluno.
2.  **Entrada de Idade** (`2_entrada_idade.alg` / `.fprg`)
    * Algoritmo para leitura e validação de dados relacionados à idade do usuário.
3.  **Controle de Estoque** (`3_estoque.alg` / `.fprg`)
    * Algoritmo voltado para a simulação de controle ou verificação de itens em um estoque.
4.  **Currículo** (`4_curriculo.alg` / `.fprg`)
    * Algoritmo para coleta e exibição de dados pessoais e profissionais (formato de currículo).
5.  **Empréstimo** (`5_emprestimo.alg` / `.fprg`)
    * Algoritmo estruturado para simular a validação ou o cálculo de um empréstimo.
6.  **Bônus de Compra** (`bonus_compra.alg` / `.fprg`)
    * Algoritmo (exercício bônus/final) para calcular benefícios, descontos ou bônus com base em uma compra realizada.

---

## 🚀 Como executar os projetos

Para testar e visualizar os algoritmos, você precisará ter os softwares instalados em sua máquina:

**Para os Pseudocódigos (`.alg`):**
1. Faça o download e abra o **Visualg**.
2. Vá em `Arquivo` -> `Abrir` e selecione o arquivo desejado.
3. Pressione `F9` para executar o código.

**Para os Fluxogramas (`.fprg`):**
1. Faça o download e abra o **Flowgorithm**.
2. Clique no ícone de pasta (Abrir) ou vá em `Arquivo` -> `Abrir` e selecione o arquivo desejado.
3. Pressione o botão de `Play` (ou a tecla `F5`) para executar o fluxo.
